# Lovee - วันครบรอบ (Anniversary Website)

เว็บไซต์วันครบรอบความรัก ที่ทำงานเสถียรและปลอดภัย

## ✨ ฟีเจอร์

- 🔐 **ล็อคหน้าจอ** - ป้อนรหัส PIN 4 หลักเพื่อปลดล็อค
- ⏱️ **นับวันที่คบกัน** - แสดงจำนวนวัน ชั่วโมง นาที วินาที
- 📸 **แกลเลอรี่รูปภาพ** - สไลด์รูปภาพแบบ interactive
- 🎬 **วิดีโอ** - เล่นวิดีโอที่คบกัน (รองรับหลายไฟล์)
- 🎵 **เพลง** - เล่นเพลงโปรดอัตโนมัติ (ไฟล์ MP3 หรือลิงก์)
- 🎂 **เค้กวันเกิด** - เป่าเทียนและ confetti effect
- ⚙️ **แอดมิน** - จัดการรูปภาพ วิดีโอ เพลง และข้อความ

## 🔧 การแก้ไขจากเวอร์ชันเดิม

### ปัญหาที่แก้ไข:
1. ✅ **แก้ CSS syntax error** - ลบ `});` เกินใน `@keyframes spin`
2. ✅ **แก้ตัวแปร PIN** - ใช้ `inputPin` แยกจาก `configData.pin` เพื่อหลีกเลี่ยงการชนกัน
3. ✅ **เพิ่ม null checks** - ตรวจสอบ DOM elements ก่อนใช้งาน
4. ✅ **ลบ resize handle logic** - ลบโค้ดที่พยายาม querySelector `.resize-handle` ที่ไม่มีใน DOM
5. ✅ **เพิ่ม error handling** - ทำให้ฟีเจอร์ทั้งหมดทำงานเสถียรแม้ไม่มี element

### การปรับปรุง:
- ✨ โค้ดเสถียรกว่า ไม่ crash บน error
- 📱 ทำงานได้ดีบน mobile และ desktop
- 🎯 ฟีเจอร์ทั้งหมดทำงานได้อย่างเสถียร

## 🚀 การใช้งาน

1. เปิดไฟล์ `index.html` ในเบราวเซอร์
2. ป้อนรหัส PIN (ค่าเริ่มต้น: `0604`)
3. ดูเนื้อหา: รูปภาพ วิดีโอ เพลง และเค้กวันเกิด
4. คลิกไอคอน ⚙️ ที่มุมบนซ้ายเพื่อเข้าสู่โหมดแอดมิน

## ⚙️ การตั้งค่า

### GitHub Token
ต้องการ GitHub token ที่มีสิทธิ์ read/write ไปยัง repository ของคุณ

### Config
ข้อมูลการตั้งค่าจะถูกเก็บใน `assets/config.json`:
```json
{
  "message": "ข้อความหลัก",
  "startDate": "YYYY-MM-DD",
  "pin": "0604"
}
```

### โฟลเดอร์ GitHub
- `images/` - เก็บรูปภาพ
- `assets/video/` - เก็บวิดีโอ (video1.mp4 - video5.mp4)
- `assets/audio/` - เก็บเพลง (main.mp3 หรือ music_link.txt)
- `assets/config.json` - เก็บการตั้งค่า

## 📋 ความต้องการ

- เบราวเซอร์ที่รองรับ HTML5 (Chrome, Firefox, Safari, Edge)
- GitHub repository ที่เป็นส่วนตัว
- GitHub token สำหรับการอัปโหลด
